## Quick start

1. Clone this repo using `git clone https://github.com/reactexcel/diagnosis.git`
2. Move to the appropriate directory: `cd diagnosis`.<br />
3. Run `npm install` in order to install dependencies.<br />
4. Run `npm start` to see the app at `http://localhost:3000` .<br />

Now you're ready to rumble!

> Please note that this boilerplate is **production-ready and not meant for beginners**! If you're just starting out with react or redux, please refer to https://github.com/petehunt/react-howto instead. If you want a solid, battle-tested base to build your next product upon and have some experience with react, this is the perfect start for you.
